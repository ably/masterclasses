# Ably Masterclasses

Ably's DevRel team runs a masterclass series with a new episode every month. These masterclasses aim to explain various realtime concepts and teach the audience how to build fun and useful realtime applications using Ably's realtime libraries, that are production-ready and fully scalable.

You can find more info about this series on our [website](https://pages.ably.com/webinar/masterclass/)
